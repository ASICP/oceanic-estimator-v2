# Dolphin v2 Calculator - Agentic Team Orchestration

**Version:** 2.0
**Date:** December 18, 2025
**Product:** Dolphin (AI Agent Team Assembly & Orchestration)

---

## Executive Summary

### Objectives
The Dolphin v2 Calculator provides pricing and margin analysis for the Dolphin service, which focuses on **AI Agent Team Assembly**. It allows clients to configure digital workforces, enabling "Team-as-a-Service".

### Key Differentiators
- **Dynamic Team Assembly**: Instantly spin up specialized agent teams (Research, Coding, Legal).
- **Inter-Agent Collaboration**: Built-in protocols for agents to critique and improve each other's work.
- **Human-in-the-Loop**: Seamless handoff protocols.
- **Secure Sandbox**: Agents run in isolated environments with controlled access.

---

## Pricing Model

### Tiers
1. **Starter (Small Team)**
   - Targeted at individuals or small startups.
   - Up to 5 Active Agents.
   - Basic Templates.

2. **Professional (Department)**
   - Growing teams.
   - Up to 20 Active Agents.
   - Advanced Collaboration Patterns.
   - Shared memory.

3. **Enterprise (Organization)**
   - Full scale automation.
   - Unlimited Agents.
   - Custom Tools & Private VPC.

### Cost Drivers (COGS)
- **Inference**: Token consumption by agents (thinking + output).
- **Compute**: Runtime environment for agent code execution.
- **Storage**: Agent state and long-term memory.

---

## Competitors
- **CrewAI Enterprise**: Strong on structure, expensive enterprise support.
- **Microsoft AutoGen**: Powerful but complex, mostly self-hosted focus.
- **Zapier Central**: Good integrations, limited complex reasoning.
- **SmythOS**: Visual builder, per-run pricing.

---

## ROI Methodology
ROI is calculated based on "Full-Time Employee (FTE) Equivalent" savings.
- **Productivity Gain**: Speed of execution vs human.
- **Cost Avoidance**: Comparison of Agent Cost vs Human Salary for same tasks.
